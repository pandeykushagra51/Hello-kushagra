Following steps are there to start the server:

  • go to folder C++ Server
  • Run the command "g++ main.cpp"
  • An executable file with name a.out will be created
  • Run executable file with command "./a.out"
  • The server will start on port 8080
  • Go to browser and see the output after calling appropriate url

Following steps are there to create the server:

  • made a system call to get file descryptor of socket
  • Bind system call is made to bind socket to PORT 8080
  • Listen system call with max_connection 10,000 is made to allow listening upto 10,000 requests concurrently
  • Run executable file with command "./a.out"
  • Accept system call is made so that the request can be take from listening queue
  • Fork system call is made create a child process and send back the HTML file

Following features are implemented

  • Proper error check at every step
  • Listening upto 10,000 request concurrently
  • On any request a html file is sent
